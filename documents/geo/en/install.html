ActiveScriptRuby 1.6.0.1

Ruby 1.6.1, Win32OLE 0.1.4, RScript 1.0.7.0, GRScript 1.0.7.0, RubyWIn 0.0.2

ActiveScriptRuby Installation Procedure

If you would like to update package, Please UNINSTALL Previous Version From Control-Panel before installation. Due to Microsoft Installer Specification. Otherwise, you may reinstall previous version again.
  1. If you are currently using Windows2000, then skip to the next.
      for NT and 9x users.
    • You need Microsoft Windows Installer Update.
    • Download from here.
  2. Download All-In-One package from here (about 700K).
    • This package contains the latest ActiveScriptRuby (RScript.dll and GRScript.dll).
    • This package also contains Ruby(MSWin)1.6.1 binary and Win32OLE binary.
    • And RubyWin for Window Environment (It's also from Mr. Suketa).
  3. Then, double-click ActiveScript.MSI from Explorer.
  4. Now, you are ready to start program with Ruby !

ActiveScriptRuby Uninstallation Procedure

  1. Setting-ControlPanel-Application... you may find "ActiveRuby".
  2. Click "Delete"
  3. Done.

Some Pakcage Information

This package contains tcltkutl.dll (Ruby's Expansion Library). It was built with Tcl83.
